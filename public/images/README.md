# Assets Folder

Place your portfolio assets here:

## Required Images

- `profile.jpg` or `profile.png` - Your professional headshot
- `logo.png` - Your logo or personal brand mark (optional)

## Optional Images  

- Project screenshots
- Gallery photos
- Certificate images
- Award photos
- Event photos

## Recommended Sizes

- Profile photo: 400x400px (square)
- Logo: 200x200px
- Gallery images: 1200x800px
- Certificates: High resolution

## Supported Formats

- JPG/JPEG
- PNG
- WebP
- SVG (for logos/icons)
